# KindaFax
Work in progress fax-like messaging system, using recept printers.

## Introduction
The goal of this project is to create a messaging system, using recept printers.

The project is split into phases, where the first phase is to create a open-source hardware design,
and create a client program to interface with the Discord API.

The second phase is to replace Discord with my own server API.

## Contributing
While direct code contributions are not required, you can still help the project in meaningful ways:
- Open an issue if you encounter any issues.
- Share your own hardware designs.
